Experiment 1: Binary Search on Student Records

Aim: To perform binary search on student records using structures in C++.

int binarySearch(Student students[], int size, int targetRollNo) {
    int left = 0, right = size - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (students[mid].rollNo == targetRollNo)
            return mid;
        else if (students[mid].rollNo < targetRollNo)
            left = mid + 1;
        else
            right = mid - 1;
    }
    return -1;
}
